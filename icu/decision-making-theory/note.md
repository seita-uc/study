## 欲求

欲求: 行動を起こす前提となる肉体的、精神的な緊張状態（eg. 空腹、不安）

cf. 動機（motive): ある場面で行動を起こす原因

欲求 => 刺激 => 意思決定 => 行動 => 成果 => 欲求

前提：欲求理論の共通項

1. 行動のきっかけは、欲求不満状態。欲求充足のために行動を起こす。
2. 欲求には異なる種類、あるいは低次・高次といった階層性がある
3. このような欲求は普遍的なもので、程度の差はあれ万人に共通

## 状況

- 環境: 行動主体を取り巻く客観的実在
- 状況: 行動を起こす際に行動主体が認識している環境 => 主観的

## 状況の構成要素

- 物的客体（人間以外の有体物）
- 人的客体
- 象徴的客体（法、宗教、慣習、道徳、良心、文化など）

cf. 象徴操作は人間の特徴

## 記憶と学習

- 学習： 経験の反復によって生ずる行動の持続的変容過程及びその結果
- 記憶： 過去の経験を保持し、必要に応じてそれを想起する家庭あるいは結果
- 記憶過程： 記銘・保持・想起の三段階からなる（偶発的や非意図的なこともある）

## 学習と行動

- 状況に対応する適応力の向上
- 記憶は習慣を生む：行為の定型化が可能　=> 進歩に集中
- 他社の経験をも自分のものにできる
- 将来の問題への対応

1. 交渉とは： 意思決定論的アプローチ
- 交渉(negotiation): 二人以上の人間が、協力して行動する未来の事柄について、話し合いで取り決めをかわすこと。

## 交渉の構造
- 交渉による合意の代わりとなる選択肢
- 交渉者の利害関係
- 交渉者に取っての利害の相対的重要度
=> 相手がとる可能性の高い行動を予測

## 交渉による合意の代わりとなる選択肢
- 最善の代替案(Best Alternative To a Negotiated Agreement: BATNA)
- 回答が自分のBATNAの価値を上回る=>受託
- 留保点(reservation point)
- 下回る=>拒否
=> 自分と相手のBATNAをきちんと認識することの重要性

## 交渉者の利害関係
- 交渉者が相手に表明するポジション(position: 交渉相手に対して求めるもの)
- 交渉者の利害関心(interest: ポジションの背後にある交渉者の動機)
=> 相手が本心で何を欲しているのか(利害関心)を理解することが重要

## 交渉者にとっての利害の相対的重要度
- 個々の交渉事項についての自分と相手にとっての相対的な重要性
= 自分にとって重要なものとそうでないものの順位づけの重要性

2. 交渉における取り分の要求
- 交渉域(bargaining zone): 双方の留保点の重複部分
    - 重複あり(正) => 妥協可能
    - 重複なし(負) => 妥協点なし
=> 「相手の言うことを聞くこと=>負け」と思わないこと

## 続き
- 異質性を利用した交渉: 相互補完によるパイの拡大
- 賭を利用した交渉: コンティンジェント契約(contingent contract: 起こるかどうか不確実な事態についての取り決めを盛り込んだ契約): 将来予測の違いを利用して合意に至る

4. 価値創出の道具
